üìù WEBHOOK SETUP INSTRUCTIONS (Do this after deployment)
================================================================================

After you deploy to Netlify, you'll need to create a webhook:

1. Go to: https://dashboard.stripe.com/webhooks
2. Click "Add endpoint"
3. Enter your webhook URL: https://yourdomain.com/api/stripe/webhook
4. Select these events:
   ‚úÖ checkout.session.completed
   ‚úÖ customer.subscription.created
   ‚úÖ customer.subscription.updated
   ‚úÖ customer.subscription.deleted
   ‚úÖ invoice.payment_succeeded
   ‚úÖ invoice.payment_failed
5. Click "Add endpoint"
6. Copy the "Signing secret" (starts with whsec_...)
7. Update STRIPE_WEBHOOK_SECRET in .env.local
8. Also add it to Netlify environment variables

================================================================================
üìß RESEND SETUP INSTRUCTIONS (Required for PDF email delivery)
================================================================================

1. Go to: https://resend.com
2. Sign up for a free account (allows 100 emails/day, 3,000/month)
3. Verify your domain (delawarezoning.com) or use their test domain for now
4. Go to API Keys section
5. Create a new API key
6. Copy the key (starts with re_...)
7. Add RESEND_API_KEY to .env.local

Note: For production, you MUST verify your domain to send from reports@delawarezoning.com
